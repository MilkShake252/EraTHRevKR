□使い方
FLAG:4（디버그플래그）を真にしてください
AIが방침を決定する際に得た数値を一覧表示します

□用語の説明
判定値　AIが実際に判定に使った数値です。RANDでの結果が加味されています
固定値　判定の際に固定値で加減されるもので、RANDが絡まない数値です
変動値　RAND:Xが加減されているものです。この数値がAIのランダムなぶれとなります
　　　　変動値は（X-1)で、例えばRAND:3なら、2となり、固定値より最大で2高くなる可能性があるということになります
期待値　固定値＋変動値/2です

	※実際の変動値は複数の値の合計値となるので期待値に近づきやすいはずです
	また、プラスとマイナスをそのまま발してるので数値が小さくてもブレ幅が大きい場合もあります
	（変動マイナスと変動プラスをわけとけばよかったんだけどまぁ面倒되었다ので기력が湧いたら…）
