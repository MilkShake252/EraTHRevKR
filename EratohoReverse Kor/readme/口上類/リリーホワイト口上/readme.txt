;◎구상内容
;----------------------------
;『せつめい』
;	リリーちゃんに甘々でえっちなイタズラをされちゃう구상です。
;	実際素質がいい子過ぎて、플레이ヤーがコントロールしないとなかなか酷い事してくれない気がします……
;	『마력＝精』という表現は대요정구상様から、同じ妖精なので一部認識に共通点を持たせてあります。
;
;『小ネタ』
;	・着衣상태、임신경험、男女別で適度に派生
;	・一部大ちゃん구상実装
;	・最初に選ぶ주인님/途中参加の주인님で分岐
;
; ＊個人メモ
;　 死に구상回避の為이벤트で素質付与
;　 죠교済み：죠교자が분유や항문절정できるようになる、かも（「EV13_죠교終了_K15.ERB」386行目～）
