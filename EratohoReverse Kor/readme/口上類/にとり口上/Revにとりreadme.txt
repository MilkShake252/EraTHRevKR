　eratohoRev専用の니토리구상ファイルです。
　私の中の니토리のイメージを思うがままに表現してみま했다。
　「니토리はそんなこと言わない」「誰てめぇ」……すみません。

　eratohoRevの구상フォルダにERBファイルを突っ込んでください。
