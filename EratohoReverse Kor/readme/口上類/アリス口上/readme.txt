◎はじめに
**********
이 구상은 eratohoЯeverse 1.2.12以降の追加要素に対応했다ものです
それ以前の本体バージョンでは確実に不具合が起きます

◎ゲーム모드について
**********************
*NORMAL
キャラ번호0 당신(名前の変更は可能)として엘리스に죠교される事になります
ゲームスタート後30日이 경과한 시점에서、一定の条件を満たせばエンディングとなります。
迎えるエンディングによってはそのまま終了、もしくはデータ로드の선택となり
로드しない場合はそのまま終了となります。
詳細はendingファイル内を確認して下さい。

플레이ヤーの性別毎の制限等はありません。どの性別でも十分に機能すると思います。
また[작은체형体型]の場合は基本的に「年端もいかない子供」という扱いになり、箇所によっては分岐します。
※[작은체형体型]は성교系へのマイナス보정が割りと強めだったり
죠교대상が[작은체형体型]の場合は割りと소프트、여자の場合はまぁ普通、남자だとややきつめの言い回しが多いかも知れず

독심능력がある場合は心の声が聞ける時があります

オープニングを経由しないとエンディングには至らない仕様です。
また、ある程度以上の古いデータで中途から開始했다場合は、EDに至らない可能性があります。
※（データの互換性は近いうちに失われるものと思われます）

*SANDBOX
基本的にNORMALに準じた内容。도구自動購入等の機能が実装されています。
OPは汎用OPを呼び、その他一部이벤트の呼び出しは殺してあります。
이 모드의 경우엔딩은 없습니다。

*EXTRA
죠교대상(つまり主人公)に사쿠야、죠교자（主人の場合限定）に엘리스を選んだ場合にのみ、専用の구상が呼び出されます。
대상限定구상はファイルを関数分けしており、完全に独立して機能します。
ver 1.53でエンディングを追加。途中の이벤트の선택肢によりEDまでのルートが分岐します。

詳細はextraファイル内を確認して下さい。

죠교대상に사쿠야以外を선택했다場合はOPは汎用OPを呼び、途中の整合性を損なうと思われる
幾つかの이벤트呼び出しは殺してあります。また、この場合は엔딩은 없습니다。

いずれのゲーム모드も新規スタート推奨となっています。

一部の整合性を損なうと思われるコマンドや問題アリと思われるコマンド、個人的に容認出来ないコマンドは
구상側で殺してあります。ご了承下さい。


◎バージョン概要
****************
・バリアント本体1.2.12に追加당했다금일의방침"かわいがって"に対応
・EV16,대만족보너스に分岐の追加と加筆
・限定無し구상と咲アリ구상で対応
・애무系ACT(가슴애무/입으로 애무/키스/파이즈리한다)に加筆
・절정구상(펠라で사정)の箇所に加筆

만족보너스[11]かわいがっては죠교자の호감도2000,죠교대상の호감도1000から선택可能
애무系の가슴애무、입으로 애무、파이즈리等に+보정されます
今まで発生しなかったこれらACTも만족보너스[11]を선택することによってモリモリ発生します
素質：혀기술の취득や豊乳化でさらに+보정
펠라で사정의際、구음경험50＆죠교자の상태변화"정욕"の時はディープスロートに派生
今の仕様で最も多くの구상を見られる自キャラは恐らく[작은체형][후타나리]持ちあたり

※その他更新당했다内容の詳細はversion.txtファイルを確認して下さい。
