◎ゲーム모드について
**********************
*NORMAL
キャラ번호0 당신(名前の変更は可能)として히나に죠교される事になります。
죠교대상を『남자』と했다場合のみに구상が呼び出される仕様。おんな、후타나리では구상は表示されません。
남자以外の場合、開始時の구상선택にも表示されません。
ゲームスタート後30日이 경과한 시점에서、一定の条件を満たせばエンディングとなります。
迎えるエンディングによってはそのまま終了、もしくはデータ로드の선택となり
로드しない場合はそのまま終了となります。
詳細はendingファイル内を確認して下さい。

仕様上、オープニングを経由しないとエンディングには至りません。
既存のデータで中途から開始했다場合は条件を満たしてもEDに至らない可能性があります。

*SANDBOX
基本的にNORMALに準じた内容。도구自動購入等の機能が実装されています。
OPは汎用OPを呼び、その他一部이벤트の呼び出しは殺してあります。
이 모드의 경우엔딩은 없습니다。

*EXTRA
OPは汎用OPを呼び、整合性を損なうと思われる幾つかの이벤트呼び出しは殺してあります。
またこの場合は엔딩은 없습니다。

죠교대상を남자以外に했다場合全ての모드で구상は表示されないので、ご了承下さい
また성교系(죠교자→죠교대상)は봉인してありますので、후타나리化や페니스밴드は無駄になります。

いずれのゲーム모드も新規スタート推奨となっています。
