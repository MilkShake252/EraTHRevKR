;eratohoReverse最初の説明
;────────────────────────────────────
まずはこのバリアントに흥미を示して、ダウン로드してくれてありがとうございます。

このバリアントはタイトル通り、플레이ヤーが죠교されるゲームです。
今までの逆죠교とは違って、플레이ヤーに죠교の主導権がまったくありません。
ただ죠교자が선택했다행동に反応するだけのゲームです。

たとえば당신は유카に자위してと命令されま했다。
まだ순종になってない당신は命令を従ってもいいし、거부してもいいです。
しかしここで거부を選んだら유카の不興を買うこととなり、
징계・として채찍で당신を打つかもしれません。
채찍으로 맞았다ら비명지른다か참다するかの선택も出ますが、
고통が強すぎる場合は叫ぶしかありません...

大体こんな感じのバリアントです。
자유性はかなり低いが、反応で죠교자の행동を영향することはできますし、
下손な죠교자の죠교をもっとスムーズに誘導するか、
上손な죠교자から脱走や逆襲を目指してもいいです。

いろんなバリアントを参考してますが、本来のeraとシステム的差異が多すぎるため、
ほぼ全部自分で一から書き込みま했다。そんな訳で플레이する前にこの三点はご了承してください：

（１）バグや設計のミスがたくさんあるかもしれません。
　　　致命的なバグでなくても、何か変な動きや理解できない状況が発生했다ら、
　　　遠慮なくスレやＩＲＣに報告してください。問題の解決に努力します
　　　もし自分でバグについで調べて直してくれるなら、最高の感謝を致します。

（２）システム的に大きく変更しま했다から、ぜひ説明書を見てから플레이してください。
　　　DATAフォルダに他にもゲームシステムの説明やまとめの文書を用意してるから、
　　　システムの仕組みについて疑問がある場合はそれらを見てくださればうれしいです。
　　　（DATAフォルダに入ってる文書は追記で詳しく説明します。）

　　　開発できたばかりで内容は豊富とは言えないし、設計的に思慮が발りない部分もあるだろうから、
　　　何か要望や意見があったらぜひスレやＩＲＣに書き込んで検討させてください。
　　　といいますが、これからは忙しくなるから、
　　　おそらく自分では次の例大祭が終わるまで大きな補발や改変は作れません。
　　　未完成の반항準備、조수、촉수、코스프레の処理もそれまで방치するようにします。
　　　バグの修正ならつくつく頑張りますが。

　　　そういうことで、追加機能として新しい행동や反応などを作ってくれるなら大歓迎です。
　　　勝손な調整や変更なども構いません。

（３）日本語はそれなりに自信がありますが所詮外人です。
　　　言語的に間違ってるところが多いと思いますし、
　　　文化的にも日本と異なる趣向がありそうです。もしか했다ら日本の方々に不適切な表現もあるかもしれません。
　　　もしより正しい日本語に修正してくれるなら幸いです。

（１）知ってる人はかなり少ないと思いますが、このバリアントの元ネタは
　　　「禁忌2～Hospital Taboo～」という古い죠교ゲームです。
　　　あのゲームでもっとも特別な設計は、죠교대상の가학心を煽るような죠교を重ねたらなんと、
　　　죠교開始の時に確率で「죠교대상が플레이ヤーを밀어넘어짐、そのまま죠교します」。
　　　その모드に入ると죠교メニューはヒロインが勝손に決めて、
　　　플레이ヤーは참다とか受け入れるとか限られた선택肢の中から選ぶしかありません。
　　　ゲーム自体にはいろいろ突っ込みたい所がありますが、
　　　あれこそ逆죠교ゲームのあり方だと、青かった私の心に深く残しま했다。
　　　회화をひとつの죠교分野として扱うのも、탈의がとてもしんどいのもあのゲームの영향です。
